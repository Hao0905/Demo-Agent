### 6.3 Upload Dataset

**Qua Dashboard:**


1. Vào `/agents/[id]/evaluate`
2. Drag & drop dataset JSON files
3. Cấu hình config file (tuỳ chọn)
4. Click **Upload**

**Qua API:**

```bash
# Tạo eval set

curl -X POST http://localhost:8000/agents/evaluate/sets \
  -H "X-API-Key: $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "website_content_test",
    "dataset_content": { "cases": [...], "metrics": [...] }
  }'

# Upload batch (multipart)
curl -X POST http://localhost:8000/api/agents/evaluate/sets/batch \
  -H "X-API-Key: $API_KEY" \
  -F "name_prefix=website_test" \
  -F "dataset_files=@dataset.json" \
  -F "config_file=@config.json"
```

### 6.4 Chạy Evaluation Job

**Qua Dashboard:**


1. Chọn eval set từ dropdown
2. Chọn số lần chạy (num_runs)
3. Click **Run Evaluation**
4. Theo dõi progress bar

**Qua API:**

```bash
# Tạo job

curl -X POST http://localhost:8000/agents/evaluate/{deployment_id} \
  -H "X-API-Key: $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "eval_set_id": "eval-set-uuid",
    "num_runs": 1
  }'

# Poll job status

curl -H "X-API-Key: $API_KEY" \
  "http://localhost:8000/agents/evaluate/{job_id}/status"

# Lấy kết quả
curl -H "X-API-Key: $API_KEY" \
  "http://localhost:8000/agents/evaluate/{job_id}/results?include_raw=true"
```

### 6.5 Đọc kết quả

Response gồm:

```json
{
  "job_id": "uuid",
  "status": "complete",
  "summary": {
    "total_cases": 10,
    "passed_cases": 8,
    "failed_cases": 2,
    "pass_rate": 0.8,
    "avg_score": 0.85,
    "duration_ms": 45000
  },
  "failures": ["Case 3: expected '...' but got '...'", "Case 7: timeout"]
}
```

### 6.6 API Evaluation endpoints

| Endpoint | Method | Mục đích |
|----|----|----|
| `/agents/evaluate/sets` | GET | Danh sách eval sets |
| `/agents/evaluate/sets` | POST | Tạo eval set |
| `/agents/evaluate/sets/{id}` | GET | Chi tiết eval set |
| `/agents/evaluate/sets/{id}` | PATCH | Cập nhật eval set |
| `/agents/evaluate/sets/{id}` | DELETE | Xoá eval set |
| `/agents/evaluate/validate` | POST | Validate dataset |
| `/agents/evaluate/{deployment_id}` | POST | Tạo evaluation job |
| `/agents/evaluate/{deployment_id}/jobs` | GET | Danh sách jobs |
| `/agents/evaluate/{job_id}/status` | GET | Job status |
| `/agents/evaluate/{job_id}/results` | GET | Kết quả job |
| `/agents/evaluate/{job_id}/cancel` | POST | Huỷ job |
