## 9. Danh sách API đầy đủ

Full API doc với data model: <https://agent-provider.public.rke.ai.tmtco.org/redoc>

### 9.1 Agent Management

| Method | Endpoint | Mục đích |
|----|----|----|
| GET | `/agents/` | Danh sách agents |
| POST | `/agents/v2/from-yaml` | Tạo agent từ YAML |
| POST | `/agents/validate-config` | Validate YAML config |
| GET | `/agents/{id}` | Chi tiết agent |
| GET | `/agents/{id}/status` | Status (lightweight) |
| GET | `/agents/{id}/endpoint` | URL internal/external |
| PATCH | `/agents/{id}` | Cập nhật metadata |
| PATCH | `/agents/{id}/scale?replicas={n}` | Scale replicas |
| POST | `/agents/{id}/deploy` | Deploy lên K8s |
| POST | `/agents/{id}/undeploy` | Undeploy |
| POST | `/agents/{id}/redeploy` | Redeploy |
| POST | `/agents/{id}/reload` | Hot-reload config |
| DELETE | `/agents/{id}` | Xoá agent |
| POST | `/agents/{id}/publish` | Publish lên registry |
| POST | `/agents/{id}/unpublish` | Unpublish |

### 9.2 Task API

| Method | Endpoint | Streaming | Mục đích |
|----|----|----|----|
| POST | `/task/send` | ❌ | Gửi task (non-streaming) |
| POST | `/task/stream` | ✅ SSE | Gửi task với streaming |
| GET | `/task/info/{task_id}` | ❌ | Chi tiết task |
| GET | `/task/plan` | ❌ | Agent plan |
| GET | `/task/artifact/{filename}` | ❌ | Download artifact |
| POST | `/task/cancel` | ❌ | Huỷ task |
| POST | `/task/continue` | ❌ | Tiếp tục task |
| POST | `/task/continue/stream` | ✅ SSE | Tiếp tục task (stream) |
| POST | `/task/resubscribe` | ✅ SSE | Kết nối lại stream |

### 9.3 Chat API

| Method | Endpoint | Mục đích |
|----|----|----|
| POST | `/chat/agents/{id}/conversations` | Tạo conversation |
| GET | `/chat/agents/{id}/conversations` | Danh sách conversations |
| POST | `/chat/agents/{id}/conversations/{cid}/messages` | Gửi message |
| GET | `/chat/agents/{id}/conversations/{cid}/messages` | Message history |
| DELETE | `/chat/agents/{id}/conversations/{cid}` | Xoá conversation |
